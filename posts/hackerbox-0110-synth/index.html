Hackerbox 0110 - Synth

‚Ė§ CONTENTS

I open-sourced some alternate code for Hackerbox 0110. This converts the Synthesizer from the project into a game where the objective is to keep the ball in the circular screen for as long as possible.

Unfortunately, I tried to prototype a 3D printed case for it, but ended up killing the screen on it while attempting to put on the case before I got a picture of the game actually running.

Anyway, if you have Hackerbox 0110 you should give it a try!
